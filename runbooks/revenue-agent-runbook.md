# Master Runbook: Complete 10-Agent Venture Orchestration

This runbook commands the main Claude Code session to route context sequentially through all 10 subagents to build an airtight, investor-ready venture asset package for Athlytica, Big Ice, and NRHL.

## Orchestration Pipeline

### Phase 1: Strategic Research & Positioning
1. **Initialize Research:** Call `market-signal-researcher` to audit `business-brief.md` for target audience demand patterns and data voids.
2. **Deconstruct Monetization:** Pass research vectors to `unit-economist` to map out the financial model, B2B tier pricing logic, and monetization targets.
3. **Formulate Offer:** Route the financial limits and market signals to `offer-architect` to build the unified, premium B2B positioning framework.

### Phase 2: Technical & Legal Infrastructure
4. **Engineer Data Layers:** Command `db-analytics-architect` to turn the offer mechanism and athletic parameters into a production-ready data structure or schema (e.g., JSON schema/SQL parameters).
5. **Enforce Compliance:** Pass the data structures to `legal-ip-steward` to verify data privacy safeguards (complying with Kenya's Data Protection Act) and map out the IP isolation model.

### Phase 3: Traffic Generation & Conversion Loops
6. **Map Media Strategy:** Pass the compliance parameters and positioning to `content-angle-strategist` to generate an under-15-minute authoritative YouTube video blueprint and retention map.
7. **Construct Conversion Funnel:** Route the video framework to `conversion-system-builder` to write the structural lead magnet assets and the 5-step executive email nurture sequence.

### Phase 4: Quality Assurance & Deployment
8. **Stress-Test System:** Invoke `technical-critic` to review all generated assets. Identify all logical gaps, weak data metrics, or generic filler language. Force a second-pass rewrite on any section failing standard.
9. **Verify Deployment Filepaths:** Route the polished assets to `infra-deployment-agent` to check local repository paths and ensure structural code readiness.
10. **Generate Operational Manual:** Call `league-ops-marshall` to synthesize all assets into an on-the-ground tournament or facility execution runbook.

## Consolidation Mandate
The main session must compile every clean block from all 10 subagents into a singular, cohesive delivery asset saved inside `outputs/revenue-agent-demo.md`. If an output smells like generic AI advice, reject the phase instantly.
