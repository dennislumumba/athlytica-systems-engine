Athlytica (parent umbrella includes Big Ice / NRHL) is B2B sports-data infrastructure eliminating data voids in athlete scouting, tracking, and risk/insurance pricing in emerging markets. Founder is Dennis Lumumba Mukhavani ([[user_founder_profile]]).

Core value prop (verbatim, must stay consistent across all downstream copy): "We build high-leverage performance intelligence and multi-sport taxonomy infrastructure that turns raw athletic potential into verifiable, high-value, monetizable data assets. We eliminate data voids."

Hard rule: no consumer/fan framing anywhere — this is institutional infrastructure (academies, agencies, federations, insurers, league operators), not a fan product.

Pipeline structure: this is a 10-agent sequential venture-orchestration pipeline. Order observed so far:
1. Market signal research agent (named data voids, buyer psychology, competitive wedge)
2. Unit economics / pricing agent (tier structure, success-fee mechanics, conversion targets)
3. Offer-architect (this role) — builds unified B2B positioning framework from agents 1+2
Downstream (per Agent 3's task brief): a db-analytics-architect (builds the schema next), a legal-ip-steward (compliance), then a content-angle-strategist and conversion-system-builder (video/email assets).

**Why:** Each agent's output is handed forward with no shared memory access between agents except what's written inline in the handoff message — so positioning decisions made here (see [[project_positioning_framework]]) must be self-consistent and complete since downstream agents cannot query this memory store to resolve ambiguity.
