Athlytica (aka Big Ice / NRHL) is a 10-agent sequential venture-orchestration pipeline. This agent occupies slot 6: takes positioning (Agent 3), schema artifact (Agent 4), and compliance guardrails (Agent 5) as fixed inputs, and outputs a full YouTube video blueprint + retention map for a technical demo video. Output is handed to Agent 7 (conversion-system-builder: lead magnet landing page + 5-step email sequence).

**Why:** The pipeline is strictly sequential and stateless between agents — each agent receives only the brief in its prompt, not prior conversation. So deliverables must be fully self-contained inline text, never files, since the next agent has no access to this agent's memory or file writes.

**Key fixed facts for this venture (as of 2026-07-08, may evolve in later pipeline runs — verify against the current brief before reusing):**
- Category name: Verifiable Athlete Infrastructure (VAI). Tagline: "We eliminate data voids."
- Free lead magnet: "Athlytica Scouting Passport Schema" (Apache 2.0, open source, institutional-email-gated).
- Paid ladder: Free schema -> "Data Void Audit" ($2,000-$5,000) -> "Athlytica Core" enterprise platform.
- Compliance guardrails are legally binding and specific (e.g., "pseudonymized" not "anonymized," "tamper-evident" not "immutable," status is pre-revenue so no fabricated case studies/customers — the $500K solidarity example is illustrative only).
- Core demo proof point: multi-sport taxonomy schema lets a new sport (inline hockey) be added via pure SQL INSERT with zero migration — this is the visual centerpiece of any technical demo content for this brand.

**How to apply:** If asked to produce more content for this venture (future video blueprints, scripts, follow-up assets), reuse these exact names/taglines verbatim and re-verify the compliance guardrails haven't been updated by a later Agent 5 pass before reusing old phrasing. Always keep the pre-revenue/no-fabricated-traction constraint in mind — it's the single easiest guardrail to accidentally violate when writing persuasive copy.
