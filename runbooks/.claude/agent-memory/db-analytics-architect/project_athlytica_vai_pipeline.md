Athlytica occupies "Verifiable Athlete Infrastructure (VAI)" — positioned as "what KYC did for finance, Athlytica does for sport." Work happens as a 10-agent sequential venture-orchestration pipeline; this agent (db-analytics-architect) is Agent 4, producing the production-ready data schema. Agent 1 did research (established trust/provenance as the core differentiator vs informal video/word-of-mouth scouting records). Agent 5 is legal-ip-steward, handling data-privacy (Kenya DPA) and IP licensing review of the schema this agent produces.

**Why:** The core product artifact for top-of-funnel is the schema itself — the "Athlytica Scouting Passport Schema," released open-source and gated behind institutional email. It is the literal deliverable buyers download, so it must be genuinely production-credible, not illustrative.

**Buyer segments the schema must serve** (as of 2026-07-08 brief):
1. Academy/Club (West/Central African football) — bone-age dispute defense, FIFA Article 21/Annex 5 solidarity claims (0.25%/yr ages 12-15, 0.5%/yr ages 16-23 of transfer fee).
2. Agent/Agency — timestamped first-contact/representation claims, rival-agent dispute defense.
3. Federation/NGB — aggregate pathway data for FIFA Forward Programme / CAF / Olympic Solidarity funder governance reporting.
4. Insurer/Underwriter — injury history + workload exposure as actuarial baseline.
5. NRHL/Big Ice (inline/ice hockey) — schema must be genuinely multi-sport at its core; hockey is positioned as white-space, zero-existing-standard, first-mover category-definition play.
6. Caribbean sprinting / East African distance running — standardizing fragmented PDF/informal meet results into queryable per-athlete records.

**How to apply:** Any future schema, pricing, or platform-tier work (Academy/Federation/Insurer/League enterprise editions) should stay consistent with this buyer-segment list and the VAI positioning. See [[project-scouting-passport-schema-design]] for the specific architectural decisions made in the v1.0.0 schema delivered 2026-07-08.
